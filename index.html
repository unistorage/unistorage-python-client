API¶

class unistorage.client.UnistorageClient(url, token)¶

Class that provides interface to the Unistorage API. Built on top of the Python requests library.

Parameters:
  • url – Unistorage API root URL.
  • token – Access token.
 
unistorage = UnistorageClient(
    'http://localhost:5000/', 'dc363e85965d461b81a4f78df45f636f')

Note

  • All described methods can raise UnistorageError.
  • All resource URIs returned by methods are relative.
apply_action(uri, action_name, action_args)¶

Applies action to the file with specified URI. Returns resource URI of the resulting file.

Parameters:
  • action_name – Action name.
  • action_args – Dictionary of action arguments.
apply_template(uri, template)¶

Applies template to the file with specified URI. Returns resource URI of the resulting file.

Parameters:template – Template URI.
create_archive(archive_name, files)¶

Creates archive. Returns resource URI of created archive.

Parameters:
  • archive_name – Archive name.
  • files – URIs of the files to be included in the archive.
 
archive_uri = unistorage.create_archive('images.zip', [file1_uri, file2_uri])
print unistorage.get(template_uri)
create_template(applicable_for, actions)¶

Creates template. Returns resource URI of the created template.

Parameters:
  • applicable_for – Files type for that template can be applied.
  • actions – List of tuples, each of which contains two elements: action name and action arguments.
 
resize = ('resize', {'mode': 'keep', 'w': 50})
grayscale = ('grayscale', {})
template_uri = unistorage.create_template('image', [resize, grayscale])
print unistorage.get(template_uri)
get(url, data=None)¶

Sends a GET request. Returns response dictionary.

Parameters:
  • url – Relative URL.
  • data – Dictionary to be sent in the query string.
post(url, data=None, files=None)¶

Sends a POST request. Returns response dictionary.

Parameters:
  • url – Relative URL.
  • data – Dictionary to be sent in the body of request.
  • files –

    Dictionary of the files for multipart encoding upload. It passed directly to the requests library. See requests documentation for details: http://docs.python-requests.org/en/latest/user/quickstart/#post-a-multipart-encoded-file

 
file = open('/path/to/file.doc')
unistorage.post('/', data={'type_id': 'news_thumbnails'}, files={'file': file})
upload_file(file_name, file_content, type_id=None)¶

Uploads file to the Unisorage. Returns resource URI of the uploaded file.

Parameters:
  • file_name – File name.
  • file_content – File-like object to be uploaded.
  • type_id – Type identifier for the file.
 
file = open('/path/to/file.doc')
file_uri = unistorage.upload_file('file.doc', file, type_id='resume')
print unistorage.get(file_uri)
exception unistorage.client.UnistorageError(status_code, msg)¶

Client raises this when it receives error from Unistorage API.

Parameters:
  • status_code – Response HTTP status code.
  • msg – Error message.

Example of usage¶

import time
from pprint import pprint

from unistorage import UnistorageClient


# Create Unistorage client
unistorage = UnistorageClient(
    'http://localhost:5000/', '3932da55c30740852fd2ba22dd59a382')

# Create template: resize and then grayscale
template_uri = unistorage.create_template('image', [
    ('resize', {'mode': 'keep', 'w': 50}),
    ('grayscale', {})
])

# Upload file
with open('/path/to/upchk.gif', 'rb') as f:
    file_uri = unistorage.upload_file('upchk.gif', f, type_id='qwerty')
pprint(unistorage.get(file_uri))

# Apply template to the file
modified_file_uri = unistorage.apply_template(file_uri, template_uri)

# See that template has been enqueued
assert unistorage.get(modified_file_uri)['status'] == 'wait'

# Wait some time
time.sleep(5)

response = unistorage.get(modified_file_uri)

# See that status has changed to 'ok'
assert response['status'] == 'ok'
# And template was applied
assert response['data']['extra']['width'] == 50
pprint(response)

# Create archive that contains both original and modified file
archive_uri = unistorage.create_archive('images.zip', [
    file_uri,
    modified_file_uri
])
pprint(unistorage.get(archive_uri))
